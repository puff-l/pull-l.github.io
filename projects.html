💳 Credit Card Fraud Detection
Group Project | Dec 2024
Problem: Credit card fraud causes financial losses and erodes trust. Need a model to detect rare fraud cases from 284k transactions.
Data: Kaggle ULB dataset, 30 anonymized features (V1–V28, Time, Amount), only 0.17% fraud.
Approach: Compared Logistic Regression, Decision Tree, Random Forest. Used stratified split, GridSearchCV, F1-score due to imbalance.
Outcome / Impact: Random Forest achieved F1=0.839, Precision=0.961, ROC-AUC=0.947, greatly reducing false positives.
My contribution: Implemented model training pipeline, hyperparameter tuning, and evaluation metrics.

🎮 GridWorld Maze with Q-Learning
Individual Project | Nov 2024
Problem: An agent must navigate from start to goal while avoiding walls and traps, demonstrating reinforcement learning.
Data: Custom 10×10 grid maps (5 fixed layouts) with start, goal, traps, bonuses, walls.
Approach: Tabular Q-learning with ε-greedy exploration, reward shaping, curriculum training across 4 maps + 1 unseen test map.
Outcome / Impact: Agent learns near-optimal policies; success rate increases, steps decrease. Greedy inference works on new maps.
My contribution: Full design and implementation of environment, Q-learning algorithm, Pygame UI, and training visualizations.
